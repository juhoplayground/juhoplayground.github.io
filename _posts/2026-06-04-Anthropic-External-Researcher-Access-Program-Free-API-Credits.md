## 개요

Anthropic의 External Researcher Access Program은 AI 안전(safety)과 정렬(alignment) 주제를 연구하는 외부 연구자를 지원하기 위한 프로그램이다.
핵심은 표준 Claude 모델 제품군에 사용할 수 있는 무료 API 크레딧을 제공해, 이 분야의 연구 진입 비용을 낮추는 것이다.

## 프로그램의 목적

이 프로그램은 AI 안전과 정렬이라는 우선순위가 높은 연구 분야에 종사하는 연구자를 직접 지원한다.
연구를 위한 API 비용 부담을 덜어줌으로써 해당 분야로의 진입 장벽을 낮추는 것이 목표다.

## 제공 내용과 자격 요건

대상은 AI 안전 및 정렬 연구를 목적으로 하는 연구자다.

### 제공하는 것

| 항목 | 내용 |
|------|------|
| 무료 크레딧 | 표준 Claude 모델 제품군에 대한 API 크레딧 |
| 기본 지급액 | 승인 시 기본 1,000달러 크레딧, 특수한 경우 추가 가능 |
| 사용 범위 | API 사용 전용 (웹앱 제외) |

### 제공하지 않는 것

프로그램은 제공 범위를 명확히 제한한다.

- 비공개 또는 실험 모델에 대한 접근
- 파인튜닝 API 접근
- Usage Policy 면제

즉, 무료 크레딧은 어디까지나 공개된 표준 모델을 API로 사용하는 데 한정되며, 정책 예외를 허용하지는 않는다.

## 신청과 심사

신청은 공식 신청 양식(Google Form)을 통해 진행한다.
팀 정보와 연구 주제에 대한 세부 사항을 제출하면 된다.

심사는 매월 첫 번째 월요일에 진행되며, 승인 시 통보된다.
다만 미승인된 경우 개별 피드백을 제공하기는 어렵다고 안내한다.

## 특수한 경우

일반적인 신청 외에 별도의 안내가 있는 경우도 있다.

| 상황 | 안내 |
|------|------|
| 긴급 상황 | 학회 전 크레딧 소진 등은 별도 이메일로 연락 |
| 적색팀 연구 | 정책 면제 대신 Model Safety Bug Bounty Program 추천 |
| 다수 연구자 | 단일 조직 계정을 운영하는 방식 권장 |

특히 적색팀(red-teaming) 성격의 연구를 위해 정책 면제를 원하는 경우, 이 프로그램이 아니라 Model Safety Bug Bounty Program을 이용하도록 안내하는 점이 눈에 띈다.

## 결론

External Researcher Access Program은 AI 안전과 정렬 연구자에게 표준 Claude 모델의 API 크레딧을 무료로 제공하는 지원 창구다.
기본 1,000달러 수준의 크레딧을 API 사용에 한해 지급하며, 실험 모델 접근이나 파인튜닝, 정책 면제는 포함하지 않는다.
매월 첫째 주 월요일에 심사가 이뤄지므로, AI 안전 연구를 계획 중이라면 연구 주제를 정리해 신청 양식을 제출하는 것을 고려할 만하다.
