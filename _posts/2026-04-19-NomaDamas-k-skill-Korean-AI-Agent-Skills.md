## 개요

NomaDamas가 공개한 k-skill은 한국인 사용자를 위해 특별히 설계된 오픈소스 AI 에이전트 스킬 모음집이다.
저장소 설명 그대로 "한국인인가요? 이 스킬 모음집을 다운로드 받아 두세요"가 핵심 메시지다.
별도 클라이언트 API 레이어 없이 한국 특유의 번거로운 작업을 AI 에이전트가 자동화하도록 한다.

## k-skill이란

이 프로젝트는 한국 서비스, 정부 시스템, 플랫폼과 상호작용해야 하는 한국인 사용자를 명시적으로 겨냥한다.
저장소는 "언젠가 무조건 쓸 때가 옵니다"라는 문구로 일상 활용도를 강조한다.
40개 이상의 기능을 제공하며 카테고리가 매우 광범위하다.

## 제공 스킬 카테고리

### 교통과 예약

SRT, KTX 열차 예약과 조회, 지하철 도착 정보 조회와 분실물 검색을 지원한다.

### 금융 서비스

| 항목 | 설명 |
|------|------|
| 한국 주식시장 정보 | 시세 및 종목 정보 |
| 토스증권 계좌 관리 | 계좌 조회 및 관리 |
| 중고차 시세 | 가격 조회 |
| 하이패스 영수증 | 통행료 영수증 발급 |

### 쇼핑과 커머스

쿠팡, 다이소, 마켓컬리, 올리브영의 상품 검색을 지원한다.
번개장터 같은 중고 마켓과의 통합, 배송 추적도 포함된다.

### 공공과 생활

한국 법령과 특허 검색, 부동산 실거래가, 가정 쓰레기 배출 일정, 학교 급식 메뉴 같은 행정·생활 정보를 다룬다.
한국 날씨와 미세먼지, 한강 수위, 로또 결과 확인, 맞춤법과 글자수 세기 같은 유틸리티도 제공한다.

### 엔터테인먼트

KBO 야구, K-리그 축구, LCK e스포츠 결과를 조회할 수 있다.
긱뉴스 피드 집계와 조선왕조실록 검색까지 지원하는 점이 흥미롭다.

## 설치와 사용 흐름

표준 워크플로는 명확하다.

| 단계 | 작업 |
|------|------|
| 1 | 전체 스킬 설치 |
| 2 | k-skill-setup 실행 |
| 3 | 개별 기능 사용 |

SRT, KTX, 특허, 증권, 통행료 같은 일부 스킬은 사용자 인증 정보를 요구한다.
대부분의 스킬은 무료 공개 API와 선택적 프록시 서버를 통해 동작한다.

## 결론

k-skill은 글로벌 에이전트 도구가 놓치기 쉬운 한국 특화 워크플로를 단일 패키지로 묶었다는 점에서 가치가 크다.
한국 사용자가 LLM 에이전트로 SRT 예약, 부동산 시세, 급식 메뉴 같은 일상 업무를 자동화하려 할 때 즉시 활용 가능한 출발점을 제공한다.
오픈소스로 공개되어 있어 필요한 스킬을 추가하거나 커스터마이즈하기에도 용이하다.
