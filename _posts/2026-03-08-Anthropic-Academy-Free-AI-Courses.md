## 개요

Anthropic이 Skilljar LMS 플랫폼 기반의 무료 온라인 교육 플랫폼 Anthropic Academy를 출시했다.
Claude 기본 사용법부터 API 활용, Claude Code 개발 워크플로, MCP 서버 구축, Agent Skills까지 총 13개 과정을 제공한다.
등록 후 진도 추적이 가능한 구조로 되어 있다.

## 개발자 대상 과정

### Claude Code in Action

Claude Code를 개발 워크플로에 통합하는 실습 과정이다.
실제 개발 환경에서 Claude Code를 활용하는 방법을 다룬다.

### Building with the Claude API

Anthropic 모델을 Claude API로 활용하는 전체 과정을 다룬다.
API 연동의 기초부터 실전 활용까지 체계적으로 학습할 수 있다.

### MCP 관련 과정

두 가지 과정이 제공된다.

Introduction to Model Context Protocol에서는 Python으로 MCP 서버와 클라이언트를 구축하는 기초를 학습한다.
Model Context Protocol: Advanced Topics에서는 프로덕션 MCP 서버 개발을 위한 고급 패턴을 다룬다.

### Agent Skills 입문

Introduction to Agent Skills 과정에서는 마크다운 기반의 재사용 가능한 스킬을 생성하고 공유하는 방법을 학습한다.

## 일반 사용자 및 특화 과정

다양한 대상별로 특화된 과정도 제공된다.

| 과정명 | 대상 |
|--------|------|
| Claude 101 | 일반 사용자 - Claude 일상 업무 활용법 |
| AI Fluency: Framework and Foundations | 전체 - AI 시스템과의 효과적 협업 |
| AI Fluency for Educators | 교육자 |
| AI Fluency for Students | 학생 |
| Teaching AI Fluency | 강사 |
| AI Fluency for Nonprofits | 비영리 조직 |

## 클라우드 플랫폼 통합 과정

주요 클라우드 서비스와의 통합 과정도 포함되어 있다.

Claude with Amazon Bedrock 과정에서는 AWS 서비스와의 연동 방법을 다룬다.
Claude with Google Cloud's Vertex AI 과정에서는 Google Cloud와의 통합을 학습한다.

## 결론

Anthropic Academy는 Claude 생태계를 체계적으로 학습할 수 있는 공식 교육 플랫폼이다.
개발자부터 교육자, 비영리 조직까지 다양한 대상에 맞춘 과정을 무료로 제공한다는 점에서 AI 활용 역량을 높이고자 하는 사람들에게 유용한 자원이다.
