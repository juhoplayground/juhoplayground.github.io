## 개요

Perplexity AI가 2026년 2월 25일, 범용 디지털 워커인 [Perplexity Computer](https://www.perplexity.ai/products/computer){:target="_blank"}를 공개했다.
Perplexity Computer는 현재 모든 AI 기능을 단일 시스템으로 통합하여, 사용자가 목표를 제시하면 알아서 계획하고 실행하는 범용 AI 에이전트이다.
채팅 인터페이스가 답변을 제공하고 에이전트가 작업을 수행한다면, Perplexity Computer는 수 시간 또는 수개월 동안 실행될 수 있는 전체 워크플로우를 생성하고 실행하는 시스템이다.

## 배경

Perplexity의 사명은 전 세계의 호기심에 동력을 제공하는 것이다.
프론티어 AI 모델들이 매우 유능해지면서, 이를 기반으로 구축된 제품들이 모델의 진정한 잠재력을 보여주는 데 병목 현상이 되고 있다.
Perplexity는 세계 최초의 AI 네이티브 브라우저인 Comet과 개인용 AI 에이전트인 Comet Assistant를 구축하며, 심층 연구를 업계 최고 수준으로 발전시키고 지속성 메모리와 작업 등을 통합해왔다.
새로운 모델들이 점점 더 전문화됨에 따라, 모든 최고의 기능을 단일 시스템으로 통합하여 가장 강력한 AI 활용을 가능하게 하겠다는 것이 이번 제품의 출발점이다.
또한 Perplexity가 구축하는 모든 것은 항상 모델에 구애받지(model-agnostic) 않았으며, 더 나은 결과를 위해서는 선택권과 통제권이 필수적이라는 철학을 기반으로 한다.

## 핵심 내용

### 작동 방식

Perplexity Computer는 인간 동료와 마찬가지로 소프트웨어 스택을 직접 조작한다.
추론하고, 위임하며, 검색하고, 구축하고, 기억하며, 코딩하고, 결과를 전달한다.

작업 흐름은 다음과 같다.

| 단계 | 설명 |
|------|------|
| 1단계 | 사용자가 자연어로 결과물을 설명한다 |
| 2단계 | Computer가 이를 작업과 하위 작업으로 나누고 서브 에이전트를 생성한다 |
| 3단계 | 서브 에이전트들이 웹 조사, 문서 생성, 데이터 처리, API 호출 등을 병렬로 수행한다 |
| 4단계 | 조정은 자동으로 이루어지며 작업은 비동기식으로 진행된다 |
| 5단계 | 최종 결과를 통합하여 전달한다 |

Computer가 문제에 봉착하면 이를 해결하기 위해 하위 에이전트를 생성한다.
API 키를 찾고, 보충 정보를 조사하며, 필요하다면 앱을 코딩하고, 정말로 사용자의 도움이 필요할 때만 확인을 요청한다.
모든 작업은 실제 파일 시스템, 실제 브라우저 및 실제 도구 통합에 액세스할 수 있는 격리된 컴퓨팅 환경에서 실행된다.
사용자는 다른 일에 집중하거나 수십 대의 Perplexity Computer를 병렬로 실행할 수도 있다.

### 멀티 모델 오케스트레이션

Perplexity Computer의 핵심 차별점은 지능형 멀티 모델 오케스트레이션이다.
기존 통념은 AI 모델이 범용화(commoditizing)되고 있다고 말해왔지만, Perplexity는 실제로 그 반대인 모델의 전문화를 목격하고 있다고 말한다.
각 프론티어 모델은 서로 다른 종류의 작업에 탁월하므로, 완전한 워크플로우는 모든 모델에 접근하고 이를 지능적으로 배치해야 한다.

현재 Perplexity Computer가 조율하는 모델 구성은 다음과 같다.

| 역할 | 모델 |
|------|------|
| 핵심 추론 엔진 | Claude Opus 4.6 |
| 심층 연구 (서브 에이전트 생성) | Gemini |
| 이미지 생성 | Nano Banana |
| 비디오 생성 | Veo 3.1 |
| 가벼운 작업 속도 최적화 | Grok |
| 긴 문맥 회상 및 광범위한 검색 | ChatGPT 5.2 |

Perplexity의 모델 독립적 제어 구조는 모델이 발전함에 따라 구성을 유연하게 변경할 수 있게 해준다.
사용자도 특정 하위 작업에 특정 모델을 직접 선택할 수 있다.
다양한 직무에서 토큰 예산이 점점 더 일반적인 고려 사항이 됨에 따라, 사용자는 최선의 사용 방안에 대한 선택권과 통제권을 가져야 한다.

### 주요 기능

Perplexity Computer가 제공하는 주요 기능은 다음과 같다.

| 기능 | 설명 |
|------|------|
| 백그라운드 작업 및 지속적 모니터링 | 시간이 지나도 자율적이고 지속적으로 실행하는 반복 작업 설정 |
| 병렬 리서치 및 브라우저 자동화 | 웹 검색, 브라우저 작업 자동화, 데이터 추출, 결과 종합 |
| 도구와 앱 연결 | Gmail, Slack, Notion, Calendar 등 수백 가지 서비스와 연동 |
| 생성 및 빌드 | 프로덕션 준비가 된 앱, 웹사이트, 보고서 등 생성 |
| 영구 메모리 | 세션 간 맥락 유지를 위한 영구 메모리 기능 |

브라우저 기반으로 로컬 설정 없이 보편적으로 접근 가능하며, 강력한 AI를 위한 안전한 제어 장치를 제공한다.

## 의미와 시사점

Perplexity Computer는 AI 산업의 중요한 방향 전환을 보여준다.
CEO는 "모델은 도구이고, 오케스트레이션이 곧 제품"이라며 모델 자체보다 이를 조율하는 능력에 가치를 두는 포지셔닝을 택했다.
단일 모델이나 계열에 의존하는 대신, 여러 모델의 장점을 동시에 활용하는 접근 방식은 AI 제품의 새로운 패러다임을 제시한다.
이는 특정 모델의 성능 저하에도 전체 시스템 안정성을 유지할 수 있다는 장점도 있다.

가격은 월 200달러의 Perplexity Max 구독자에게 제공되며, 월 10,000 크레딧과 20,000 보너스 크레딧(30일 유효)이 포함된다.
Enterprise Max 사용자에게도 곧 제공될 예정이다.

"컴퓨터(Computer)"라는 이름은 1757년 수학자 알렉시 클레로(Alexis Clairaut)가 핼리 혜성 예측을 위해 고용한 두 명의 "컴퓨터(계산원)"에서 영감을 받았다.
수개월 동안 밤낮으로 작업을 분담하여 핼리 혜성의 근일점을 이틀 이내의 오차로 예측한 이 이야기는, 복잡한 작업을 자율적으로 분담하고 정확성을 근본 요건으로 삼는다는 의미를 담고 있다.

## 결론

Perplexity Computer는 단순한 챗봇이나 단일 에이전트를 넘어, 19개의 AI 모델을 지능적으로 조율하는 범용 디지털 워커이다.
모델의 전문화 추세를 인정하고, 각 모델의 강점을 최대한 활용하는 오케스트레이션 접근 방식을 채택했다.
격리된 컴퓨팅 환경에서 실제 도구와 서비스에 접근하며, 수 시간에서 수개월에 걸친 워크플로우를 자동으로 실행할 수 있다.
AI가 곧 컴퓨터라는 비전 아래, 복잡한 작업을 자율적으로 분담하고 실행하는 새로운 시대를 열고 있다.
