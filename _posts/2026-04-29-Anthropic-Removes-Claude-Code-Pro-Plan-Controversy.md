## 개요

2026년 4월 21일 오후, Anthropic이 월 20달러짜리 Claude Pro 구독 플랜에서 Claude Code 접근 권한을 제거하는 정책 변경을 감행했다.
지원 문서와 웹사이트 가격 책정 페이지가 수정되어 "Pro 또는 Max 플랜"이라는 표현이 "Max 플랜"으로만 변경되었다가 이후 되돌려졌다.
Anthropic은 이를 "신규 고객층의 약 2%를 대상으로 한 소규모 테스트"라고 해명했지만, 광범위한 지원 문서 수정이 테스트라는 설명과 맞지 않는다는 비판이 제기되었다.
Ed Zitron의 "Where's Your Ed At"에서는 이 사건을 비용 절감 동기와 기업 과금 체계 전환의 연장선으로 해석했다.

## 사건 경과와 회사 입장

사건은 2026년 4월 21일 오후에 시작되었다.
Anthropic이 월 20달러짜리 Pro 구독 플랜에서 Claude Code 접근 권한을 제거했고, 지원 문서와 웹사이트 가격 책정 페이지가 수정되었다.
"Pro 또는 Max 플랜"이라는 표현이 "Max 플랜"으로만 변경된 것이 변경 사항의 핵심이다.

Anthropic 성장 담당 부사장 Amol Avasare는 이것이 신규 고객층의 약 2%를 대상으로 한 소규모 테스트라고 주장했다.
그러나 기존 Pro 사용자는 웹 앱과 CLI를 통해 여전히 Claude Code에 접근 가능했다고 한다.
논란이 확산되자 Anthropic은 웹사이트와 지원 문서 변경사항을 되돌렸다.

| 항목 | 내용 |
|------|------|
| 발생 시점 | 2026년 4월 21일 오후 |
| 변경 대상 | Pro 구독 플랜의 Claude Code 접근 권한 |
| 회사 해명 | 신규 고객 약 2% 대상 소규모 테스트 |
| 현재 상태 | 지원 문서와 웹사이트 변경 사항 롤백 |
| 기존 사용자 영향 | 웹 앱과 CLI로 계속 접근 가능했다는 증언 |

## 비판 논점

### 비일관성 문제

Ed Zitron이 제기한 핵심 비판은 비일관성이다.
지원 문서를 광범위하게 수정하면서 "소규모 테스트"라는 설명이 맞지 않는다는 것이다.
진정한 소규모 A/B 테스트라면 공식 문서 전반을 수정할 이유가 없으며, 이는 내부 결정이 이미 완료된 상태에서 테스트라는 완곡어법을 사용한 것으로 해석될 여지가 크다.

GeekNews 커뮤니티도 유사한 관점을 제기했다.
"소규모 테스트"는 이미 내부 결정이 완료된 상태의 신호로 해석되며, 공식 문서를 되돌린 것은 여론 반발에 대한 사후 대응이라는 분석이다.

### 비용 절감 동기

Zitron은 이번 조치를 비용 절감 동기와 연결한다.
The Information 보도에 따르면 Anthropic은 최근 기업 고객을 월정액 기반에서 백만 토큰당 요금 체계로 전환했다.
Pro 플랜에서 Claude Code를 제거하려는 시도는 이러한 기업 과금 체계 전환의 후속 조치로 보인다.

Avasare의 발언에서는 향후 모든 구독 계층에 변화가 올 수 있음이 시사되었다.
저자는 이것이 사용자들을 API 유료 이용으로 강제하려는 전략으로 해석한다.
월정액 무제한 구조가 GPU 용량 대비 수익성이 낮다는 판단이 깔려 있다는 관측이다.

## 커뮤니티 반응

GeekNews 사용자들은 대체로 이 조치를 소비자 기만으로 비판했다.
GPU 부족 등으로 인한 수익성 강화 추진이 의심된다는 의견이 다수였다.
OpenAI를 비판해 온 Anthropic이 결국 유사한 상업화 전략을 취한다는 지적도 있었다.

| 주요 우려 사항 | 내용 |
|-------------|------|
| 정책 일관성 | 잦은 정책 변경과 공지 불일치로 인한 신뢰 저하 |
| 수익화 동기 | GPU 용량 한계로 인한 월정액 모델 압박 |
| 토큰 할당량 | 기존 서비스 품질 저하와의 연관성 |
| 대안 전환 | Codex 등 다른 서비스로의 이동 고려 |

토큰 할당량 감소 등 기존 서비스 품질 저하와의 연관성을 우려하는 목소리도 나왔다.
다수 사용자는 이번 사건을 계기로 다른 서비스로의 전환을 검토하고 있다고 밝혔다.

## 결론

Anthropic의 이번 조치는 단발성 테스트로 보기 어려운 구조적 신호를 담고 있다.
기업 고객 과금 체계 전환과 Pro 플랜 축소 시도가 연결된다면, 이는 월정액 기반 AI 구독 모델의 경제성 한계를 보여주는 사례가 된다.
개발자에게는 단일 벤더에 종속되지 않는 도구 선택의 중요성이 부각되며, OpenClaw처럼 Anthropic API를 직접 호출하는 경로나 OpenAI Codex 같은 대안을 함께 검토할 필요가 있다.
AI 제품 구독 정책은 당분간 변동성이 클 가능성이 높으므로, 운영 파이프라인 설계 시 벤더 이동 비용을 낮게 유지하는 것이 현실적인 대응 전략이다.
